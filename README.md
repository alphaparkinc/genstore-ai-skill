# genstore-ai-skill

> GenPark AI Agent Skill — Launch and manage AI-powered online stores with [Genstore.ai](https://www.genstore.ai/)

## What This Skill Does

This skill enables AI agents to help users **launch, design, and grow online stores** using Genstore's AI-native platform. It encapsulates Genstore's full workflow — from store creation in a single prompt to dropshipping, social commerce, copywriting, analytics, and marketing automation.

## Capabilities Covered

| Domain | Details |
|--------|---------|
| **Store Launch** | Create a live storefront from a text prompt in ~2 minutes |
| **AI Agents** | Dean (Design), Genius (Strategy), Analytics, Copywriting, Marketing agents |
| **Dropshipping** | DSers/AliExpress integration, millions of products, zero inventory |
| **Print-on-Demand** | Custom branded products (mugs, shirts, posters) |
| **Social Commerce** | TikTok & Instagram native integration |
| **SEO Copywriting** | Auto-generate product descriptions from images |
| **Email/SMS Marketing** | Klaviyo, Mailchimp, Omnisend, SMS integrations |
| **Analytics** | Conversational store performance insights |
| **Migrations** | Import from Shopify or Amazon |

## Usage

Reference this skill in your GenPark agent to enable Genstore.ai capabilities:

```yaml
skills:
  - genstore-ai
```

Or prompt the agent directly:

```
"Use the genstore-ai skill to help me launch a fashion dropshipping store"
```

## Target Users

- New store owners with zero e-commerce experience
- Social creators (TikTok, Instagram) monetizing their audience
- Part-time sellers wanting low-maintenance stores
- Entrepreneurs testing new product niches quickly
