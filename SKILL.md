# GenStore.ai Skill

This skill empowers AI agents to utilize the GenStore.ai platform, an AI-native e-commerce solution that uses a team of specialized, autonomous agents rather than traditional dashboards.

## Core Capabilities: The AI Agent Team

### 1. Design Agent
- Generates professional store layouts, branding, color schemes, and typography.
- Creates motion graphics and custom visual elements based on natural language descriptions.

### 2. Product Agent
- Automates product curation and catalog management.
- Drafts SEO-optimized product listings and descriptions.
- Generates and manages high-quality product imagery and variations.

### 3. Launch Agent
- Handles final "go-live" readiness checks, including SEO baseline setup and store compliance.
- Ensures all technical prerequisites are met before the store goes public.

### 4. Analytics & Growth Agents
- Tracks customer journeys to provide actionable insights for optimizing conversion rates.
- Creates and manages dynamic promotions (e.g., discount templates).
- Automates marketing campaigns across email, SMS, and social media platforms.

### 5. Multi-Platform Integration
- Syncs the store's product catalog seamlessly across multiple sales channels: Google Shopping, Facebook Marketplace, Amazon, and TikTok.

## Usage Guidelines
When implementing this skill, agents should interact with GenStore via conversational prompts:
1. **Store Creation**: Prompt the system with the niche, brand identity, and target audience to auto-generate the storefront.
2. **Product Management**: Delegate tasks to the Product Agent to upload items or write descriptions using just the product names or basic images.
3. **Marketing**: Instruct the Campaign Agent to launch specific promotions across designated channels.

## Best Practices
- Provide clear, detailed natural language instructions to the GenStore agents.
- Leverage the adaptive learning of the platform by providing feedback to the generated outputs.
